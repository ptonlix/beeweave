# 推特风格示例

> 这个文件放置不同类型社交短内容的示例，供写作时参考。

## 锐评型

```
Claude Code 用了一个月，我删掉了。

不是因为不好用。恰恰相反，是因为太好用了，好用到我觉得自己在退化。以前写代码会先想逻辑，现在是直接描述需求让 AI 写。有一个月没自己 debug 过。

最恐怖的是，我居然也没觉得自己少了什么。

直到昨天要手写一个简单的 SQL，我卡了五分钟。五年前闭着眼睛都能写的东西。这根拐杖，我有点不敢用了。
```

## 分享型

```
用了三年的 Notion，今天终于全部迁移走了。

不是什么工具的问题，是我发现我一个产品经理的笔记里，90% 都是"先存着以后看"。存了三年没看过任何一篇。

现在我的笔记系统就两个：一个 txt 文件记当天的想法，一个 Obsidian 仓库放长期知识。

简单到蠢，但反而每天真的在用了。工具越轻，人的重量就越重。
```

## 吐槽型

```
今天看到一个"AI 创始人"发的视频，整个人麻了。

他的产品叫"AI 自动生成小红书爆款"，月费 199，功能就是一个套壳的 ChatGPT 加一个固定的 prompt 模板。

这玩意如果我愿意，15 分钟能做出来。卖 199。

最骚的是他评论区一群人在问链接。我寻思了一下我没寻思明白。可能这个时代的产品不是卖给懂的人的。
```

## 洞见型

```
最近想明白一件事：AI 时代，最稀缺的不是会问问题的人，是会闭嘴的人。

会问问题的人，AI 帮你问。GPT 给你的 follow-up question 比你自己想到的还多。

但会闭嘴的人，知道什么时候不听 AI 的。知道答案不能替代思考。

你知道 AI 什么都告诉你，你还能选择自己去想，这才是真的稀缺。
```

## 叙事型

```
昨天在咖啡店，旁边坐了个程序员在教他妈用 ChatGPT。

他妈第一句话是"你跟我讲讲它是什么"。他想了半天，说"你试试问它你最想问的事"。

他妈问"我老伴走了三年了，我该怎么走出来"。

那个程序员当场愣住了。我也愣住了。

我们天天聊 prompt engineering，但从来没有人教过这个。Prompt 是方法，不是目的。目的是那个你一直忍着没问的问题。
```

## 吹水型

```
深夜随便聊聊。

这两天疯狂刷 AI 新闻，感觉每天都有一百个新东西出来。但停下来想想，自己真正每天都在用的，其实就那几个。Claude Code 写代码，ChatGPT 当搜索引擎用，Perplexity 查资料。

剩下的都是噪音。

所以现在看到"炸裂""颠覆""革命"的标题直接划走。真的好东西不需要炸裂，它会自己渗透进你的生活里。静水流深的那种。
```

## Thread 示例

```
第一条
我做了三年 AI 内容，今天想明白了 4 件以前没想明白的事 👇

🔹
第一件，AI 不会取代你，但会用 AI 的人会取代不会用的人——这句话是废话。真正的区别不是会不会用，是你把 AI 当助手还是当同事。助手是你下指令它执行，同事是你跟它讨论、它启发你、你反过来质疑它。当助手的人效率提升 30%，当同事的人效率不是翻倍，是重新定义了自己的工作。

🔸
第二件，内容创作者的护城河从来不是技巧。以前觉得是选题角度、是文笔、是信息差。现在 AI 在这几件事上都比你强。但有一件事 AI 永远做不了——亲自下场。你买了 9.9 的 DeepSeek 自己去试、你凌晨三点爬起来写稿、你敢说"我用了三个月，我踩了这些坑"。AI 编不出这些。

🔹
第三件，真实比完美值钱一万倍。之前我写东西总是想润色到滴水不漏，但回头看效果最好的那些推文，都是半夜情绪上来一口气打完直接发的。有错别字、有语法不通、有情绪过激。但读者感受到了我是一个活人，不是一个内容的机器。

🔸
第四件，保持好奇本身就是一种能力。不是每个人看到新东西都会兴奋的。很多人看到 AI 第一个反应是"我要被取代了"，第二个反应是关掉。但如果你看到新工具第一反应是"我能用它玩什么"，你已经在前面了。不是技能比别人好，是你没有关掉那扇门。

回到第一件。这四件事说到底是一件事：AI 时代，你最大的壁垒不是你知道什么，是你是什么样的人。
```
