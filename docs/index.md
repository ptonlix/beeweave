# BeeWeave

BeeWeave is an agent-native creation workbench for building a creative data
flywheel: collect source material, create with agents, distill durable
knowledge, reuse that context, collect better material, and create the next
piece.

![BeeWeave overview](assets/beeweave.png)

```text
collect -> create -> distill -> reuse context -> collect better material -> create the next piece
```

## What BeeWeave Gives You

- A `workbench/` for rough inputs, captures, sources, and drafts.
- A `vault/` for durable Markdown knowledge that agents can search and reuse.
- Shared skills for ingesting sources, querying knowledge, and syncing useful
  context from active projects.
- Bootstrap files for multiple agents so they can work against the same
  knowledge system.

BeeWeave is not just agent memory. It is a repeatable workflow for turning raw
material and finished work into a reusable context layer.
